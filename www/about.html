About this App

This app was created as an entry for the FBLA Mobile Appilcation Development event. In creating this app, I used the following free tools:

  • CSV to JSON - to convert the quiz data into JSON
  • Excel - to create the quiz data
  • Github - to store and track my software versions, store publicly-accessible source code, and host the web app.
  • GitKraken - a GUI tool to work with Github.
  • Phonegap - to convert the web pages into an app
  • Phonegap - to convert the web pages into an app
  • SlickQuiz - to generate and display the quizzes
  • Visual Studio Code - to edit the app's pages
  • XCode - to convert the app into an IOS app

This open-source app is hosted on Github and is under GPLv3-licensed. The binary releases can be accessed here, https://github.com/ctdoan/FBLA-quizzes/releases. Please read the installation instructions for your distribution.

Created as a HTML5, CSS, and Javascript webapp, this app was converted into a mobile app using Phonegap. The original webapp is accessible at https://ctdoan.github.io/FBLA-quizzes/www/.
