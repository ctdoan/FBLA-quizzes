About this App

This app was created as an entry for the FBLA Mobile Appilcation Development. In creating this app, I used the following free tools:

  • CSV to JSON - to convert the quiz data into JSON
  • Excel - to create the quiz data
  • Github - to store and track my software versions
  • Mobirise - to quickly mock up a responsive framework for the app
  • Phonegap - to convert the web pages into an app
  • SlickQuiz - to generate and display the quizzes
  • Visual Studio Code - to edit the app's pages
  • XCode - to convert the app into an IOS app
